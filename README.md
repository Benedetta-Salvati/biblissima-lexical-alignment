# biblissima-lexical-alignment

Alignment of lexical and linguistic reference resources for medieval French, supporting interoperability of annotated corpora and NLP for historical languages.

## Description

This repository documents research conducted within the framework of the Biblissima+ programme at the École nationale des Chartes – PSL, focusing on the alignment of lexicographic and linguistic reference resources used for the annotation of medieval French corpora.

Historical corpora are often encoded according to different editorial traditions, normalization practices, and lexical reference systems. This heterogeneity limits comparability between corpora, complicates reuse of annotated data, and constrains the development of robust NLP tools for languages characterized by high orthographic and linguistic variation.

The objective of this work is to explore methods for aligning lexical resources in order to:

- facilitate interoperability between annotated corpora
- improve comparability of linguistic annotations
- enable integration of lemmatized data into shared research infrastructures
- support reproducible computational analysis of historical texts
- contribute to the development of NLP tools for medieval French and other highly variable languages

## Repository structure

This repository consolidates exploratory materials from several research environments, including:

### lexical resources
Structured lexical data derived from existing lexicographic reference systems used for medieval French.

### lemmatization workflows

Scripts and notebooks supporting experimentation with lemmatization and normalization strategies for historical French.

These materials illustrate:

- preprocessing pipelines
- testing of lexical coverage
- exploration of variant forms

### alignment experiments

Materials related to the alignment of heterogeneous lexical resources, including:

- correspondence tables between lexical reference systems
- alignment hypotheses
- exploratory scripts for resource reconciliation
- intermediate data structures

### data modelling

Documentation of data structures used to represent lexical information in interoperable formats.

Includes reflection on:

- structuring lexical entries
- linking lemma identifiers across resources
- preparing reusable datasets

## Methodological perspective

The research adopts an exploratory and iterative approach, combining philological expertise with digital methods in order to:

- identify correspondences between heterogeneous lexical resources
- support interoperability between annotated corpora
- contribute to reproducible research workflows
- enable integration of historical corpora into shared NLP infrastructures
